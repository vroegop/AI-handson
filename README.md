# AI Hands-on Presentation

Marp-based slide deck.

## Setup

```bash
npm install
```

## Develop

Live preview server with hot reload:

```bash
npm start
```

## Build

```bash
npm run build       # HTML
npm run build:pdf   # PDF
npm run build:pptx  # PowerPoint
```

Edit `slides/presentation.md` to author slides. Separate slides with `---`.

For VS Code, install the [Marp for VS Code](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode) extension for inline preview.
